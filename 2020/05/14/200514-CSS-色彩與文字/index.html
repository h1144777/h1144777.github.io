200514-CSS-色彩與文字

發表於 2020-05-14 更新於 2020-05-19 分類於 複習 閱讀次數： Disqus：

在網頁的呈現中，色彩及文字是不可或缺的，在這篇文章中會介紹使用CSS來處理網頁上的色彩及文字。

色彩

色彩數值

關於指定顏色的方式可以有以下幾種:

  1. RGB或RGBA:
    透過紅藍綠的組成來表現色彩。例如:rgb(100,100,90)
    rgba則是多了一個不透明度(alpha)，這個屬性在CSS3之後加入，數值介於0.0~1.0。例如:rgba(100,100,90,0.5)

  2. HEX碼:
    以一個#字號為開頭，接著六位數的碼來組成的色碼。例如:#ee3e80

  3. 顏色名:
    使用瀏覽器預設的顏色名稱。例如:DarkGray

  4. HSL、HSLA屬性:
    這個屬性為CSS3加入的顏色屬性，使用方法為hsl或hsla為開頭，緊接著是放在括弧內個別的數值，依序為色相、彩度、明度、不透明度(alpha，若需要此數值，開頭使用hsla，不需要則使用hsl)。例如:hsla(0,100%,100%,0.5)

  5. opacity不透明度:
    在CSS3中加入了這個屬性，作用是用來指定標籤及所有子標籤的不透明度，數值介於0.0~1.0之間，例如 opacity: 0.15 即為15%的不透明度。

關於顏色數值的取得，可使用美工軟體(例如illustrator、CorelDraw、photoshop)檢視並取得，也可參考網路上眾多的選色器並獲取數值，例如Colors Tutorial。

色彩的功能屬性

color: 此屬性可指定標籤內部的文字顏色。

background-color:css將每個HTML標籤視為獨立的方框，這個屬性就是用來指定該方框的背景色。

文字

控制文字的外觀屬性有分為兩種，一種是直接影響字體以及字本身的外觀屬性，另一種則是不限字體，對文字都有相同效果的屬性，以下是關於CSS處理文字效果的介紹。

字型與字體

font-family:這個屬性可以指定CSS內文字的字型，屬性的值為字型名稱，訪問網站的使用者電腦中需要有安裝相對應的字體才能夠正常顯示，可以指定一串字型，若電腦中沒有相對應第一順位的字體，則會往下一個指定字體尋找。 例如:font-family: Arial, Verdana, sans-serif, “Courier New”,。+

font-size:此屬性用來指定字體大小，常用的單位有:px , % , em 等等。

@font-face:這個屬性可以指定連結到字型檔的路徑，連結到此網頁的用戶電腦內若沒有此字型，就會進行下載並顯示在頁面。
但要注意的是這個方式允許字型下載到用戶電腦，所以關於字體授權的部分要特別小心。
關於font-face用法範例如下:

font-family:這個屬性用來指定字型名稱，在接下來的CSS中可以使用這個屬性內的名稱作為font-family的值。

src:這個屬性為字型的路徑。

format:這個屬性用來指定字型格式，關於字型有多種格式，可參考以下表格
副檔名 format指定方式
.woff format("woff")
.ttf、.otf format("opentype")
.eot format("embedded-opentype")
.svg、.svgz format("svg")
