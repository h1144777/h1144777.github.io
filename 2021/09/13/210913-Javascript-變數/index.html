變數簡單來說，就是一個可以放資料的空間，它的用途就是用來儲存資料，以及要讀取資料時可以從變數內讀取資料出來進行運算。

變數名稱也被稱為識別字，雖然說可以自由命名，但是還是有些需要注意的地方:

識別字名稱只能使用英文大小寫、數字、底線，以及錢符號 $ 作為命名，而且識別字不能用數字作為開頭，大小寫不同會視為不同的識別字，以及JS有一些保留字是不能作為識別字名稱，最後則是習慣會使用駝峰式命名法，用來方便閱讀，比如說 firstName ，就是單字與單字中間不分開，在第二個單字的第一個字母使用大寫。

JS版本ES6的保留字如下:

break byte case catch
char class const continue
debugger default delete do

export extends false final
finally float for function
goto if implements import
in instanceof int interface
let long native new
null package private protected
public return short static
super switch synchronized this
throw throws transient true
try typeof var void
volatile while with yield
